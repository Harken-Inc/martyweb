# About Hightail

Hightail is an AI-powered platform designed for search analysis and brand discovery.

## Our Mission

We help businesses and researchers understand search landscapes through advanced AI analysis.

## Key Features

- **Multi-Model Analysis**: Compare results across Claude, GPT, and Groq
- **Brand Extraction**: Identify brand mentions with exact positioning
- **Cost Tracking**: Monitor token usage and expenses
- **Source Attribution**: Track credibility and citations

## Technology

Built with modern web technologies including:

- Python Flask backend
- React TypeScript frontend
- PostgreSQL database
- Multiple AI model integrations
